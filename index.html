Instructions

    Reset

  • Button: "All"

    - Reset all elements on the canvas.

  • Button: "Mass"

    - Reset the size and position of the main element.


    Main Element

  • Directional Buttons

    - Up, Down, Left, Right keys to move.


    Surrounding Elements

  • Add Element (w/ Cursor)

    - Clicking on the screen adds an element at the location of the cursor.

  • Button: "Populate"

    - Adds 15 more elements to the canvas.

  • Button: "Clear"

    - Clears the canvas of all surrounding elements.

  • Toggle: "Anti-Gravity"

    - Reverses the gravity of all elements. Objects will now repel one another.
